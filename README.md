# Flowcat Homebrew Tap

Official Homebrew tap for [Flowcat](https://github.com/eunjae-lee/flowcat) Mac app.

## Installation

```bash
brew tap eunjae-lee/tap
brew install --cask flowcat
```

Or install in one command:

```bash
brew install --cask eunjae-lee/tap/flowcat
```

## Updating

```bash
brew upgrade flowcat
```

## Uninstall

```bash
brew uninstall --cask flowcat
```
